# manual

## build
```
git clone https://github.com/oluceps/telegramGroupHistory2mongodb.git
cd telegramGroupHistory2mongodb
cargo build --release
```
## usage  

```shell
cd target/release  
to_mongodb [json file] [mongodb url]
```
